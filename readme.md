# Leaselock Cert Report CLI

A CLI for cert reports

## Installation

### Clone the Repo

```
git clone git@github.com:djheru/cert-report.git
```

### Update .env file

```
cp .env.example .env
# Update to use the correct API URL
```

### Install Dependencies

```
cd cert-report && yarn install
```

### Link the CLI

```
yarn link
```

### Generate a Report

```
cert-report 1,2
```
